
# Veritas Legal AI

**Empowering legal professionals with AI-driven insights from verified legal documents.**

Veritas Legal AI is a professional platform designed to analyze official laws, court rulings, contracts, and other verified legal sources. The platform generates accurate legal reports, highlights risks, and provides actionable insights for lawyers and legal teams. It includes secure case management and a context-aware AI assistant to streamline legal workflows.

## Platform Overview

Veritas Legal AI is designed to help lawyers and legal teams:

* Extract key clauses and summaries from complex contracts
* Compare multiple contracts efficiently
* Detect potential legal risks and compliance gaps
* Access verified legal sources for research
* Automate time-consuming review tasks and calculate ROI

The platform prioritizes **accuracy, security, and usability** to support modern legal practice.

---

## Core Features

### Reliable Document Analysis

* Extract key clauses from contracts and legal documents
* Summarize long regulations and laws
* Support multiple document formats, including PDFs and scanned documents

### Legal Risk Assessment

* Highlight ambiguous or weak clauses
* Identify compliance gaps and potential liabilities
* Assign severity ratings for each risk

### Verified Legal Research

* AI-powered answers based on official laws and court rulings
* Citations included for every insight
* Semantic search across all documents

### Smart ROI Estimation

* Estimate time and cost savings per case
* Automate initial review and research workflows

## Advanced Case Management

### Partner Dashboard

* Manage Active, Pending, and Archived cases
* Quick overview of alerts, deadlines, and case metrics

### Individual Case Interface

* **Secure Login per Case** (unique access credentials)
* Tabs for:

  * Documents
  * People involved
  * Notes & highlights
  * Risk Analysis
  * Context-Aware AI Legal Assistant

### Context-Aware AI Assistant

* Provides case-specific analysis
* Can summarize documents, extract clauses, and flag risks
* Fully aware of all documents and notes within a case

## Technology Stack

* **Frontend:** React / React Native (for mobile)
* **Backend:** Node.js / Express
* **Database:** Firebase (Firestore)

* **Authentication:** Firebase Auth / OAuth2

## Getting Started

### Prerequisites

* Node.js >= 18
* Yarn or npm
* Firebase project credentials

### Installation

```bash
git clone https://github.com/<your-username>/Veritas-Legal-AI.git
cd Veritas-Legal-AI
yarn install    # or npm install
```

### Configuration

* Set up Firebase credentials in `.env`
* Configure OpenAI API key in `.env`
* Initialize Firestore database

### Run the App

```bash
yarn dev       # or npm run dev
```
